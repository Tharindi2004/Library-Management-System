# Library Management System (Windows Forms)

## Description
This is a C# Windows Forms application developed as a group project.
The system allows librarians to manage books, issue books, and handle returns.

## Features
- Add new books
- Issue books to members
- Return books
- Dashboard for navigation

## Technologies Used
- C#
- .NET Framework
- Windows Forms
- SQL Server

## How to Run
1. Open the solution file in Visual Studio
2. Restore NuGet packages
3. Press F5 to run the application
